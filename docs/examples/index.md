# Examples

Task-focused guides for the main features of cw-commons. Each page is a self-contained walkthrough
with copy-pasteable code.

- [Config Loading](config-loading.md) — load and validate a YAML config with `ConfigManager`.
- [Commands](commands.md) — register commands with `BaseCommand`.
- [SQLite Data Store](store-sqlite.md) — the bundled write-behind key-value store.
- [MongoDB Data Store](store-mongo.md) — the same API backed by MongoDB.
- [Player Data Manager](player-data.md) — per-player data that flushes on quit.
- [Auto-Flush Task](store-auto-flush.md) — periodic background flushing.

For the full API, see the [Javadoc](https://jitpack.io/com/github/CrimsonWarpedcraft/cw-commons/latest/javadoc/).
